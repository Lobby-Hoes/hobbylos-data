# Hobbylos Data

Apollo GraphQL based API

## Docker
### Quick Start
- ``git clone https://github.com/Lobby-Hoes/hobbylos-data.git``
- ``docker compose up --build``

## Node
### Requirements

- node
- npm

### Quick Start
- Node Environment auf "development" setzen ```$env:NODE_ENV="development"```
- ``npm install``
- ``node index.js``
